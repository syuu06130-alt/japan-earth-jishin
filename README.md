# 🗾 JISHIN.WATCH — 日本地震リアルタイム監視システム

> **リアルタイムで日本の地震を監視するWebアプリケーション**
> 気象庁(P2P地震情報)・USGSの無料公開APIを使用。APIキー不要・サーバー不要で動作します。

## 📋 機能一覧

| カテゴリ | 機能 |
|---|---|
| 🗺️ 地図 | Leaflet.js + 標準/衛星写真/地形図 切替（すべてAPIキー不要） |
| ⚡ リアルタイム更新 | **毎秒〜1分** の範囲で自動更新間隔を変更可能 |
| 🌐 波紋アニメーション | 地震発生箇所から波紋が広がる視覚表現 |
| 🔍 地名検索 | 都道府県・主要都市・トラフ名で検索→地図移動 |
| ⛶ 大画面モード | ヘッダーのボタンで全画面モニター表示のON/OFF切替 |
| 🚨 緊急地震速報 | M6.0以上で全画面アニメーション警告（「大地震が発生しています」） |
| 🌊 津波警報 | 津波情報検知で全画面アニメーション警告（「直ちに避難を」） |
| 🤖 監視Bot | 新着地震の自動検知・ログ記録・ブラウザ通知・音声アラート |
| 📊 統計グラフ | マグニチュード分布・時間別件数・震源深度の可視化 |
| 📰 ニュースフィード | 気象庁発表をリアルタイムで記事化して表示 |
| 📜 過去の大地震 | 関東大震災〜能登半島地震まで主要地震＋動画リンク集 |
| 🛟 防災情報リンク | 気象庁・内閣府・ハザードマップ等への外部リンク集 |
| 🔥 密度ヒートマップ | 地震多発地域を視覚化 |
| 🏔️ 断層帯・プレート境界 | 主要活断層とプレート境界のオーバーレイ表示 |
| 📱 レスポンシブ対応 | スマートフォン・タブレット対応 |

## 🚀 セットアップ

### ローカルで起動

```bash
git clone https://github.com/YOUR_USERNAME/jishin-watch.git
cd jishin-watch

# 静的ファイルサーバーで起動（どちらか）
python3 -m http.server 8080
npx serve .
```

ブラウザで `http://localhost:8080` を開く

> **注意**: `index.html` を直接ダブルクリックで開く（`file://`）と、ブラウザのセキュリティ制限でAPI取得が失敗する場合があります。必ずローカルサーバー経由で開いてください。

### GitHub Pages で公開

1. GitHubにリポジトリを作成し、本フォルダの中身をすべてpush
2. `Settings → Pages → Source: main branch（ルート）` に設定
3. `https://YOUR_USERNAME.github.io/jishin-watch/` でアクセス可能

## ⚙️ 主な設定（js/config.js）

```javascript
BOT: {
  FETCH_INTERVAL: 30000,        // 初期の自動更新間隔（UIから毎秒〜1分に変更可）
  ALERT_MAG_BANNER:    5.0,     // この規模以上でバナー表示
  ALERT_MAG_EMERGENCY: 6.0,     // この規模以上で緊急全画面警告
}
```

UI上の「自動更新間隔」セレクトボックスから **毎秒更新** に変更できます（ただしAPI提供元への負荷を考慮し、通常利用では30秒以上を推奨）。

## 🧪 緊急アラートのテスト方法

ブラウザの開発者コンソール（F12）で以下を実行すると、実際の地震を待たずにアニメーションを確認できます。

```javascript
AlertsModule.testEmergency()   // 大地震警告アニメーションをテスト表示
AlertsModule.testTsunami()     // 津波警報アニメーションをテスト表示
```

## 📡 使用データソース（すべて無料・APIキー不要）

| ソース | 説明 |
|---|---|
| [P2P地震情報](https://www.p2pquake.net/) | 気象庁発表の地震・津波情報をリアルタイム配信 |
| [USGS](https://earthquake.usgs.gov/fdsnws/event/1/) | 全世界の地震データ（日本周辺で補完利用） |

地図タイルは **CartoDB（標準）／Esri World Imagery（衛星）／OpenTopoMap（地形）** をすべてAPIキー不要で利用しています。

## 🌊 震度スケール対応表

| 震度 | 区分 | 状況 |
|---|---|---|
| 7   | 激震 | ほとんどの建物が倒壊 |
| 6強 | 烈震 | 多くの建物が倒壊 |
| 6弱 | 烈震 | 立つことが困難 |
| 5強 | 強震 | 家具が転倒する |
| 5弱 | 強震 | 棚の食器が落ちる |
| 4   | 中震 | 眠っている人が目覚める |
| 3   | 弱震 | 棚の食器が揺れる |
| 2   | 軽震 | 電灯などが揺れる |
| 1   | 微震 | 屋内で静かな人が感じる |

## ⌨️ キーボードショートカット

| キー | 操作 |
|---|---|
| `R` | データを手動更新 |
| `F` | 日本全体を表示 |
| `Esc` | モーダル・大画面・警告アニメーションを閉じる |
